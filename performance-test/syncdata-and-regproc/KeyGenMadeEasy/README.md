# Contains
* Pythons is required as a pre-requisite.
* 5000 pre-generated key pair and certificates are kept in "generated_keys.7z" folder. Extract the same before use. Update the "keyPath" variable within the Jmeter script under "user defined variables".
* To create more, Open "justKeys.py" in a text editor. Update the value of NUM_KEYS.
* Execute "justKeys.py" with python.
